# Fake News Detector

A Machine Learning-based web application that classifies news articles as **Fake** or **Real** using Natural Language Processing (NLP) and Machine Learning algorithms.

## Project Overview

The Fake News Detector helps identify misleading or false news articles by analyzing the text content using NLP techniques and a trained machine learning model.

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Fake-News-Detector.git
cd Fake-News-Detector
```

### Install Dependencies

```bash
pip install flask pandas numpy scikit-learn
```

### Run Application

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```

## Working Process

1. Enter the news article or headline.
2. Click Detect.
3. The text is preprocessed using NLP.
4. The trained model predicts whether the news is:
   - Fake
   - Real

## Dataset

The project uses the **news.csv** dataset for training and testing the machine learning model.
